# inner 'Item'

makes a request to mongodb Item collection

Properties:
Name Type Description
find method

uses query to find the user specified db data
sort method

takes static priority value (for now) and a dynamic value formed using the value orderBy formed using the get_sort_option method
skip method

uses the start_ndx number to begin searching at #x
limit method

limits the number of returned items to the user specified limit
lean method

strips the results down to only the db collection array data
Example
rows = await Item.find(query).collation({locale: "en" })
.sort([ priority, ...orderBy])
.skip(start_ndx).limit(limit)
.lean();
